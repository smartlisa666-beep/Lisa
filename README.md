## 📋 프로젝트 개요

하하매스 교육 플랫폼은 선생님과 학생들을 위한 디지털 학습 관리 시스템입니다. 교과서 학습, 과제 관리, 평가 실시, 수업 자료 공유 등 교육 활동에 필요한 모든 기능을 통합적으로 제공합니다.

### 🎯 주요 목표
- **효율적인 학습 관리**: 한 곳에서 모든 교육 활동 관리
- **직관적인 인터페이스**: 누구나 쉽게 사용할 수 있는 UI/UX
- **실시간 피드백**: 학생 학습 현황 즉각 파악
- **데이터 기반 교육**: 학습 데이터 분석 및 활용

## ✨ 주요 기능

### 1. 🏠 홈 (index.html)
- **가장 최근 수업**: 최신 수업 정보와 자료 바로가기
- **학습 현황 캘린더**: 2025년 12월 기준 일별 학습 활동 시각화
- **학생 리더보드**: 5명 학생의 학습 성취도 및 참여도 순위
  - 하트(참여도), 완료 과제 수 표시
  - hahamath066-01 ~ 05 학생 관리
- **사용자 프로필**: 선생님 정보 (hahamath066-t, 1학년별)
- **빠른 작업**: 과제/평가 생성, 수업 시작 바로가기
- **최근 활동 피드**: 실시간 학습 활동 알림

### 2. 📚 교과서 (textbook.html)
- **슬라이드 뷰어**: 교과서 내용을 슬라이드 형식으로 표시
- **썸네일 네비게이션**: 좌측 슬라이드 목록으로 빠른 이동
- **인터랙티브 퀴즈**: O/X 퀴즈 기능
  - 예시: "43902는 사만 삼천구백이다" (정답: X)
- **관련 자료**: 현재 학습 내용 관련 추가 자료 제공
- **페이지 네비게이션**: 이전/다음 슬라이드 이동 (1/5 형식)

### 3. 📝 과제 (assignment.html)
- **과제 목록 관리**: 카드 형식의 과제 목록 표시
- **상태별 필터링**: 전체/예상/진행 중/종료
- **단원별 필터**: 1~6단원 개별 선택
- **과제 정보 표시**:
  - 제목, 기간, 작성자
  - 제출 현황 (예: 3/5명)
  - 남은 기간 또는 완료 상태
- **과제 카드 예시**:
  - "초등 수학 4-1_1차시별 학습 평가_6.관계와 규칙_7~8차시"
  - 진행 중 과제 4개, 종료 과제 2개
- **액션 버튼**: 수정, 삭제, 상세보기

### 4. 📊 평가 (evaluation.html)
- **평가 목록 관리**: 시험 및 평가 목록
- **출제 방식 구분**: AI 출제 / 작성 출제
- **상태별 관리**: 전체/예상/진행 중/종료/일시 저장
- **평가 정보 표시**:
  - 문제 수 (12~20개)
  - 응시 현황 (예: 2/5명)
  - 평균 점수 (종료된 평가)
- **평가 카드 예시**:
  - "초등 수학 4-1_1차시별 학습 평가_4.삼각형_2~3차시"
  - AI 출제 2개, 작성 출제 4개
- **일시 저장 기능**: 작성 중인 평가 임시 저장

### 5. 📁 수업자료실 (materials.html)
- **추천 자료 캐러셀**: 최근 수업 관련 자료 가로 스크롤
- **카테고리 필터**: 교과서/과제/평가
- **단원별 필터**: 1.큰 수 ~ 6.관계와 규칙
- **난이도 필터**: 상/중/하
- **정렬 옵션**: 최신순/인기순/좋아요순
- **자료 정보 표시**:
  - 좋아요, 조회수, 다운로드 수
  - 날짜, 단원, 난이도
- **인터랙션**: 좋아요, 다운로드, 상세보기

## 🎓 사용자 가이드

### 선생님을 위한 가이드

1. **대시보드 확인**
   - 홈 페이지에서 오늘의 일정 확인
   - 학생 학습 현황 모니터링
   - 최근 활동 피드 확인

2. **수업 진행**
   - 교과서 메뉴에서 수업 자료 열기
   - 슬라이드 형식으로 수업 진행
   - 퀴즈로 즉각적인 이해도 확인

3. **과제 관리**
   - 과제 메뉴에서 새 과제 생성
   - 제출 현황 실시간 확인
   - 필터로 과제 상태 관리

4. **평가 실시**
   - 평가 메뉴에서 평가 생성
   - AI 출제 또는 직접 출제 선택
   - 응시 현황 및 결과 확인

5. **자료 활용**
   - 수업자료실에서 필요한 자료 검색
   - 카테고리, 단원, 난이도로 필터링
   - 좋아요 표시로 유용한 자료 북마크
